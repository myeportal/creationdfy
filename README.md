# Creation DFY

Creation DFY is a root-level workspace project for `creationdfy.com`.

## Positioning
A warm, done-for-you AI setup and digital marketing agency for people with strong ideas but no clear way to build them into something valuable.

## Offer
- One offer only
- **$1,750 annual engagement**
- Also payable in **12 monthly installments**
- **30-day money-back guarantee, no questions asked**

## Brand Direction
- warm
- caring
- hopeful
- capable
- light, appealing design
- dreams-to-fruition / seed-to-tree growth metaphor

## Source Material
This project was rebranded from the workspace's idea-to-creation / selfmademoneyplan white-label materials and adapted into a DFY agency model for `creationdfy.com`.

## Files
- `index.html` — main landing page
- `assets/` — reused/rebranded imagery
- `vercel.json` — static deployment config

## Deployment Notes
- Intended GitHub repo: `https://github.com/myeportal/creationdfy`
- Intended production host: Vercel
- If the GitHub repo is already connected to Vercel, pushing `main` should trigger deployment automatically.
